## Overview
This repository documents my cybersecurity learning journey through TryHackMe labs. These modules focus on building foundational knowledge of computer systems, operating systems, and security concepts that are essential for understanding cybersecurity, system administration, and future security-focused roles.

## Modules Completed

## Module 4: Computer Fundamentals

### Topics Covered
- **Inside a Computer System**
  - Learned how computer components work together, including the CPU, memory, storage, and hardware resources.
  - Developed an understanding of how systems process information and execute tasks.

- **Computer Types**
  - Explored different types of computers and their roles in personal, business, and enterprise environments.

- **Client-Server Basics**
  - Learned the fundamentals of client-server architecture and how devices communicate over networks.
  - Gained an understanding of how services, requests, and responses function within a network environment.

- **Virtualization Basics**
  - Learned how virtual machines allow multiple operating systems to run on one physical machine.
  - Explored the benefits of virtualization and its relevance in cybersecurity environments.

- **Cloud Computing Fundamentals**
  - Learned the basics of cloud infrastructure, services, and deployment models.
  - Developed an understanding of cloud security concepts and shared responsibility.
