# Network Fundamentals – TryHackMe

## Overview
Completed foundational networking training focused on how devices communicate across networks and the basic protocols used in cybersecurity and IT environments.

## Topics Covered
- IP addresses and MAC addresses
- OSI and TCP/IP models
- Common networking protocols (HTTP, HTTPS, DNS, DHCP)
- Ports and network communication
- Routing and switching basics

## Skills Developed
- Understanding how data travels across networks
- Identifying common protocols and their functions
- Building networking knowledge essential for cybersecurity analysis
- Strengthening troubleshooting and technical reasoning skills

## Takeaways
This module provided a strong understanding of networking concepts that are critical in cybersecurity, especially for identifying suspicious traffic, analyzing connections, and understanding how systems communicate.
