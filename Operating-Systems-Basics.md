## Module 5: Operating System Basics

### Topics Covered
- **Operating Systems Introduction**
  - Learned the role of operating systems in managing hardware, applications, users, and system resources.
  - Built foundational knowledge of how operating systems impact security.

- **Windows Basics**
  - Explored Windows environments, system organization, and common administrative concepts.
  - Learned basic security considerations within Windows systems.

- **Linux CLI Basics**
  - Practiced navigating Linux systems through the command line.
  - Learned essential commands for file management, permissions, directories, and system interaction.
  - Built foundational skills used in cybersecurity, penetration testing, and security operations.

- **Operating System Security**
  - Learned fundamental security practices including authentication, access control, permissions, and system updates.
  - Developed awareness of how vulnerabilities and misconfigurations can impact system security.
